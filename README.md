Project di repo ini fokus pada latihan **logika backend, CRUD sederhana, dan pengolahan data menggunakan database**.

## 🛠 Tech Stack

- **PHP Native**
- **MySQL**
- **XAMPP / Apache**

Project ini fokus pada **logika backend**, sehingga tampilan masih sederhana.

---

## 🚀 Cara Menjalankan Project

1. Clone repository

```
git clone https://github.com/fudzah31/web-programming-practice.git
```

2. Pindahkan project ke folder

```
xampp/htdocs/
```

3. Jalankan **Apache** dan **MySQL** di XAMPP.

4. Buka di browser

```
localhost/web-programming-practice/basic-web-crud
```
